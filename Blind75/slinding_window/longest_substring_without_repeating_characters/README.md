# Longest Substring Without Repeating Characters

## 問題概要
文字列 `s` の中で、重複文字を含まない最長の部分文字列の長さを返す。

## 計算量
- 時間計算量: O(n) — `right` ポインタが左から右へ1回走査（`left` の移動にループは消費しない）
- 空間計算量: O(min(m, n)) — ハッシュマップに登録するのは登場するユニーク文字のみ（`m` は文字種の数、実質 O(1)）

## アプローチ

**スライディングウィンドウ + インデックスマップ** を使う。

- `char_index_map[c]`：文字 `c` が最後に登場したインデックスを記録
- `right` を右に伸ばし、重複が見つかったら `left` を「重複位置の次」にワープさせる

## ポイント

**`char_index_map[s[right]] >= left` の条件が必要な理由**

例：`"abba"` を処理する場合

1. `a(0), b(1), b(2)` と進み、2番目の `b` で重複 → `left` を `2` に移動
2. 最後の `a(3)` に到達したとき、マップには `a` のインデックス `0` が残っている
3. この条件がないと `left` が `0+1=1` に**後退**してしまいバグになる
4. `>= left` を確認することで「窓の外にある古い重複は無視する」安全弁になる
