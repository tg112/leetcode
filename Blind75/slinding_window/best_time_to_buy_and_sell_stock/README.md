# Best Time to Buy and Sell Stock

## 問題概要
株価の配列 `prices` が与えられる（`prices[i]` は i 日目の価格）。1回だけ売買して得られる最大利益を返す。  
利益が出ない場合は `0` を返す。

## 計算量
- 時間計算量: O(n) — 配列を1回スキャン
- 空間計算量: O(1) — `min_price` と `max_profit` の2変数のみ

## アプローチ

**過去の最安値を追跡しながら、今日売った場合の利益を毎日計算する。**

1. `min_price` を初日の価格で初期化
2. 2日目以降、各日について:
   - `min_price = min(price, min_price)` で最安値を更新
   - `max_profit = max(max_profit, price - min_price)` で利益を更新
3. 最終的な `max_profit` を返す

## ポイント
- 「買い」は必ず「売り」より前でなければならない → 左から右へ1回ループするだけで時系列の制約を自然に満たす
- 株価が単調に下がるケースでは `max_profit` は `0` のまま返る（取引しない選択）
- 貪欲法（Greedy）の一種：各ステップで「今日売った場合の最大利益」を即座に評価する
