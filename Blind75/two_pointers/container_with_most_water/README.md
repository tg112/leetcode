# Container With Most Water

## ポイント

「幅が狭くなっていくんだから、ボトルネック（低い壁）を更新しにいかないと、面積が大きくなる見込みは万に一つもない！」

ex) heights = [1, 8, 6, 2, 5, 4, 8, 3, 7] 

初期状態:

left = 0 (高さ 1)  
right = 8 (高さ 7)  
幅 = 8、高さ = min(1, 7) = 1 ⇒ 面積 = 8  

次の移動:

left の方が低い（1 < 7）ので、left を右に動かします（left = 1 へ）。  

※この瞬間、高さ1の棒を使った他の組み合わせ（右端を内側に動かしたもの）は、面積が8未満になることが確定しているので、調べる必要がなくなります。

ステップ2:

left = 1 (高さ 8)  
right = 8 (高さ 7)  
幅 = 7、高さ = min(8, 7) = 7 ⇒ 面積 = 49  
今度は right の方が低い（8 > 7）ので、right を左に動かします（right = 7 へ）。
