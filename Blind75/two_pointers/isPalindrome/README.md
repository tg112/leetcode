# Valid Palindrome

## 問題概要
文字列 `s` が与えられる。英数字のみを考慮し（大文字小文字は無視して）、回文かどうかを判定する。

## 計算量

**解法1（Two Pointers）**
- 時間計算量: O(n)
- 空間計算量: O(1) — 追加のメモリなし

**解法2（文字列反転）**
- 時間計算量: O(n)
- 空間計算量: O(n) — クリーニング済み文字列のコピーを保持

## アプローチ

**解法1：Two Pointers（推奨）**  
`left` と `right` を両端に配置し、内側へ向かって比較する。

1. 英数字でない文字（記号・空白）はスキップ（`isalnum()`）
2. `s[left].lower() != s[right].lower()` なら即 `False`
3. 一致すれば両ポインタを内側へ
4. `left >= right` になるまで繰り返し、問題なければ `True`

**解法2：文字列反転**  
`s` を英数字だけにフィルタリングして小文字化し、`cleaned == cleaned[::-1]` で判定する。  
コードが短いが、O(n) の追加メモリが必要。

## ポイント
- `isalnum()` — 英数字（a-z, A-Z, 0-9）であれば `True`、記号・空白は `False`
- `s[::-1]` — Python のスライスで文字列を反転する構文（`step=-1`）
- 空文字列は回文として扱われる（`left >= right` が最初から成立）
