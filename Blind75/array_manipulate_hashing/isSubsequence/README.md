# Is Subsequence

## 問題概要
文字列 `s` が文字列 `t` の部分列（subsequence）であるかを判定する。  
部分列 = 元の文字の**順序を保ちながら**、いくつかの文字を取り除いて作れる文字列。

## 計算量
- 時間計算量: O(n) — `t` の長さ n を1回走査
- 空間計算量: O(1)

## アプローチ

**Two Pointers** — `s` と `t` を同時にスキャンする。

- `s[itr1] == t[itr2]` のとき、両ポインタを進める（`s` の文字を消費）
- 一致しない場合は `itr2` だけ進める（`t` を読み飛ばす）
- ループ終了後、`itr1 == len(s)` なら `s` のすべての文字が順番通りに見つかった → `True`

## ポイント
- 順番が大事 — ポインタを左から右にしか動かさないことで、順序の制約を自動的に満たす
- コードには3つの解法がある:
  1. **while ループ**（2ポインタ）: 基本形
  2. **for ループ**: `t` を for で回し、`s` のポインタ `i` を進める
  3. **`iter` + `all`**: `iter(t)` を使い、`s` の各文字が `t` の中に順番に存在するかを1行で確認
